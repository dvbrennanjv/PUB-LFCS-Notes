Task A Filesystem Drill
----------------------------------------------------------------
In your home directory:
Create a directory structure:
~/lfcs_practice/
    logs/
    configs/
    scripts/

Inside logs/, create 3 empty files:
app.log
db.log
access.log

Move access.log into configs/ and rename it to nginx-access.log.
Copy db.log into configs/ (keeping the same name).
Delete app.log.


REVIEW
When moving use relative paths by ~ or .. if deeper in a directory
